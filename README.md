## Electric lime  -  color palatte


## Neobrutalism - Design Principles

- **Hard borders** - thick, solid outlines on every component
- **Flat shadows** - offset box shadows, no blur, high contrast
- **Raw typography** - bold, heavy
- **No gradients** - flat fills only
- **Intentional asymmetry** - structured

## Getting Started

No build tools. No installs. Just open it.

```bash
# Clone the repo
git clone https://github.com/Ux49/portfolio.git

# Open in browser
open index.html
```

Or simply drag `index.html` into your browser.
